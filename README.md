# Account-Management-System
Simple Account-Management System. QA Individual Exercise  

Justification of Technology Used:

Application Server: Wildfly- Familiarity with platform, full Java EE 7 support & Java 8 preview mode, Free/LGPL licensing, HTTP/2 support

Build Tool: Maven- Familiarity with platform, simple dependency packaging, integrates nicely with other systems e.g. Sonar.

Mock tetsing framework: Mockito- Familiarity with platform, mock objects for automated unit testing of methods that require external services i.e databases.
